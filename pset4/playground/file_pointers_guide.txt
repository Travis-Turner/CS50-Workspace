Some other file pointer operations in stdio.h
fgets() - gets a full string from a file
fputs() - writes a full string to a file
fprintf() - Writes a formatted string to a file
fseek() - allows you to rewind of fast-forward within a file
ftell() - Tells you at what (position) byte you are within a file
feof() - tells you if you've read to the end of a file
ferror() - indicates if there was an error working with the file
